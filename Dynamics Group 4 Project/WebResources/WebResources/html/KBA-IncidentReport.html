Reporting an Incident

How to Report an Incident:

  1. Login to your account at Contoso Mortgages.
  2. Along the website's main menu bar located at the top of page, select Report Incident .
  3. Incident Report Fields:
    1. Mortgage: Select an existing Mortgage this Incident applies to.
    2. Title: The Incident's title can be anything you like that helps to generally describe the Incident.
    3. Description: A more detailed description of the Incident.
    4. Priority: Specify 'High', 'Normal', or 'Low' to denote the urgency of the incident.
    5. High Priority Reason: This field only appears if you selected 'High' as the Priority level. This field is meant to be the rationale for this incident being a high priority.
    6. Type: Specify the type of the incident
